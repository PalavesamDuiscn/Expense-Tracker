# 💸 Expense Tracker

A clean, dark-themed expense tracker web app built with **HTML, CSS, and JavaScript**.
No frameworks. No dependencies. Just open `index.html` in a browser.

---

## ✨ Features

- Add expenses with description, amount, category, and date
- Live stats — Total Spent, This Month, Top Category
- Visual bar chart broken down by category
- Transaction list with per-item delete and clear all
- Data persists in the browser via `localStorage`
- Fully responsive — works on mobile

---

## 📁 Project Structure

```
expense-tracker/
├── index.html   → Page structure
├── style.css    → All styling & animations
├── script.js    → App logic (add, delete, render, localStorage)
└── README.md    → This file
```

---

## 🚀 How to Run

1. Clone the repo:
   ```bash
   git clone https://github.com/YOUR_USERNAME/expense-tracker.git
   ```
2. Open `index.html` in any browser — that's it!

---

## 🌐 Deploy on GitHub Pages

1. Push code to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch → `/ (root)`
4. Your app will be live at `https://YOUR_USERNAME.github.io/expense-tracker`
